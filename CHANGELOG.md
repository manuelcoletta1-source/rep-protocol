# Changelog

All notable changes to the Reconstructible Event Protocol (REP) draft will be documented in this file.

## draft-coletta-rep-00

Initial experimental draft of the REP specification.

Defines the minimal reconstructible event model:

Decision  
Cost  
Trace  
Time

Introduces the basic event unit structure:

event_id  
decision  
cost  
trace  
time_start  
time_end
